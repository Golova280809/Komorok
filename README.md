## 📖 Что это?

**Komorok** — это мой личный сайт, открытый для всех. Он объединяет в себе:

- **Komorium** — ручную энциклопедию знаний о географии, истории, технологиях и Забайкальском крае.
- **Блог** — заметки об учёбе, программировании и развитии проекта.
- **Стихи и творчество** — мои произведения.

Весь сайт написан с нуля на **чистом HTML, CSS и JavaScript**, без использования конструкторов или готовых шаблонов.

## ✨ Особенности

- 🌓 **Тёмная и светлая тема** — переключается и запоминает ваш выбор.
- 🔍 **Мгновенный поиск** по всей энциклопедии (по названиям и описаниям).
- 📂 **Сворачиваемые разделы** — как в Яндекс.Дзене, для удобной навигации.
- 🤖 **Автоматические счётчики** — скрипты на Python сами обновляют статистику на сайте через GitHub Actions.
- 📱 **Адаптивный дизайн** — одинаково хорошо выглядит на телефоне и компьютере.

## 🧠 Интеллектуальная начинка (Автоматизация)

Komorok — не просто статичный сайт. Я написал Python-скрипты, которые «оживляют» его:

1.  **`count_articles.py`**: Автоматически подсчитывает количество статей в папке `Komorium/` и обновляет счётчик на главной странице.
2.  **`update_stats.py`**: Ежедневно пересчитывает, сколько дней прошло с момента основания проекта, и обновляет это число в футере.

GitHub Actions запускает их по расписанию, так что сайт меняется сам, без моего участия. Весь код скриптов находится в папке `scripts/`. 

## 🚀 Быстрый старт (если захотите запустить у себя)

1.  **Клонируйте репозиторий**
    ```bash
    git clone https://github.com/golova280809/Komorok.git
    ```
2.  **Откройте `index.html`** в браузере — проект не требует сборки или установки зависимостей.
3.  **Чтобы заработали автообновления**, поместите папку `.github/workflows` в корень вашего репозитория и включите GitHub Actions в настройках.

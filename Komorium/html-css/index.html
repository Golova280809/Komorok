Что такое HTML

📰 HTML-CSS: Что это такое и с чем его едят?

Введение

Каждый сайт в интернете — это HTML и CSS. Когда ты открываешь любую страницу в браузере, ты видишь результат работы этих двух технологий. Даже самые сложные порталы, соцсети и поисковики начинались с HTML и CSS.

HTML отвечает за структуру — что где находится. CSS отвечает за внешний вид — цвета, шрифты, расположение элементов.

HTML — язык разметки (скелет сайта)

CSS — каскадные таблицы стилей (дизайн сайта)

HTML → структура
CSS  → дизайн
JS   → поведение

Что такое HTML?

HTML (HyperText Markup Language) — язык разметки гипертекста. Это основа любой веб-страницы. С его помощью браузер понимает, где заголовок, где текст, где картинка или ссылка.

HTML использует теги — специальные команды в угловых скобках:

<p>Это параграф текста</p>
<h1>Заголовок</h1>
<a href="ссылка">Ссылка</a>
<img src="картинка.jpg">

Теги бывают двух видов:

  • Парные — <тег>...</тег> (с закрывающим слэшем)
  • Одиночные — <тег> (без закрывающего слэша, например <img>)

Основные теги HTML

Вот теги, которые ты будешь использовать постоянно:

Заголовки (h1-h6)

<h1>Главный заголовок</h1>
<h2>Подзаголовок</h2>
<h3>Мелкий заголовок</h3>

Текст

<p>Параграф текста</p>
<span>Короткий текст внутри строки</span>
<strong>Жирный текст</strong>
<em>Курсив</em>

Ссылки и картинки

<a href="https://site.com">Ссылка</a>
<img src="foto.jpg" alt="Описание картинки">

Списки

<ul>
    <li>Маркированный пункт</li>
</ul>

<ol>
    <li>Нумерованный пункт</li>
</ol>

Что такое CSS?

CSS (Cascading Style Sheets) — каскадные таблицы стилей. Если HTML это скелет сайта, то CSS — это его одежда и макияж. С помощью CSS можно изменить цвет, шрифт, размер, расположение элементов и многое другое.

CSS состоит из селекторов и свойств:

селектор {
    свойство: значение;
    свойство: значение;
}

Пример:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

p {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

📌 Селектор — указывает, к какому тегу применить стиль (h1, p, .class, #id)

📌 Свойство — что именно изменить (color, font-size, margin...)

📌 Значение — значение свойства (blue, 16px, 10px...)

Основные свойства CSS

Вот самые популярные свойства, которые ты будешь использовать:

Текст и шрифт

/* Цвет и шрифт */
color: #ff0000;           /* Цвет текста */
font-size: 16px;          /* Размер шрифта */
font-family: Arial;      /* Шрифт */
font-weight: bold;       /* Жирный */
text-align: center;      /* Выравнивание */

/* Отступы между строками */
line-height: 1.5;

Фон и границы

/* Фон */
background: #f0f0f0;     /* Цвет фона */
background-image: url('bg.jpg'); /* Картинка */

/* Границы */
border: 1px solid black;
border-radius: 8px;       /* Скругление углов */

Отступы и размеры

/* Внешние отступы */
margin: 20px;             /* со всех сторон */
margin-top: 10px;         /* сверху */

/* Внутренние отступы */
padding: 15px;
padding-left: 10px;       /* слева */

/* Размеры */
width: 300px;            /* ширина */
height: 200px;           /* высота */

💡 margin — отступ снаружи элемента

💡 padding — отступ внутри элемента

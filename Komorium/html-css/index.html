Что такое HTML

📰 HTML-CSS: Что это такое и с чем его едят?

Введение

Каждый сайт в интернете — это HTML и CSS. Когда ты открываешь любую страницу в браузере, ты видишь результат работы этих двух технологий. Даже самые сложные порталы, соцсети и поисковики начинались с HTML и CSS.

HTML отвечает за структуру — что где находится. CSS отвечает за внешний вид — цвета, шрифты, расположение элементов.

HTML — язык разметки (скелет сайта)

CSS — каскадные таблицы стилей (дизайн сайта)

Что такое HTML?

HTML (HyperText Markup Language) — язык разметки гипертекста. Это основа любой веб-страницы. С его помощью браузер понимает, где заголовок, где текст, где картинка или ссылка.

HTML использует теги — специальные команды в угловых скобках:

<p>Это параграф текста</p>
<h1>Заголовок</h1>
<a href="ссылка">Ссылка</a>
<img src="картинка.jpg">

Теги бывают двух видов:

  • Парные — <тег>...</тег> (с закрывающим слэшем)
  • Одиночные — <тег> (без закрывающего слэша, например <img>)

Основные теги HTML

Вот теги, которые ты будешь использовать постоянно:

Заголовки (h1-h6)

<h1>Главный заголовок</h1>
<h2>Подзаголовок</h2>
<h3>Мелкий заголовок</h3>

Текст

<p>Параграф текста</p>
<span>Короткий текст внутри строки</span>
<strong>Жирный текст</strong>
<em>Курсив</em>

Ссылки и картинки

<a href="https://site.com">Ссылка</a>
<img src="foto.jpg" alt="Описание картинки">

Списки

<ul>
    <li>Маркированный пункт</li>
</ul>

<ol>
    <li>Нумерованный пункт</li>
</ol>

Что такое HTML

📰 HTML-CSS: Что это такое и с чем его едят?

Введение

Каждый сайт в интернете — это HTML и CSS. Когда ты открываешь любую страницу в браузере, ты видишь результат работы этих двух технологий. Даже самые сложные порталы, соцсети и поисковики начинались с HTML и CSS.

HTML отвечает за структуру — что где находится. CSS отвечает за внешний вид — цвета, шрифты, расположение элементов.

HTML — язык разметки (скелет сайта)

CSS — каскадные таблицы стилей (дизайн сайта)

HTML → структура
CSS  → дизайн
JS   → поведение

Что такое HTML?

HTML (HyperText Markup Language) — язык разметки гипертекста. Это основа любой веб-страницы. С его помощью браузер понимает, где заголовок, где текст, где картинка или ссылка.

HTML использует теги — специальные команды в угловых скобках:

<p>Это параграф текста</p>
<h1>Заголовок</h1>
<a href="ссылка">Ссылка</a>
<img src="картинка.jpg">

Теги бывают двух видов:

  • Парные — <тег>...</тег> (с закрывающим слэшем)
  • Одиночные — <тег> (без закрывающего слэша, например <img>)

Основные теги HTML

Вот теги, которые ты будешь использовать постоянно:

Заголовки (h1-h6)

<h1>Главный заголовок</h1>
<h2>Подзаголовок</h2>
<h3>Мелкий заголовок</h3>

Текст

<p>Параграф текста</p>
<span>Короткий текст внутри строки</span>
<strong>Жирный текст</strong>
<em>Курсив</em>

Ссылки и картинки

<a href="https://site.com">Ссылка</a>
<img src="foto.jpg" alt="Описание картинки">

Списки

<ul>
    <li>Маркированный пункт</li>
</ul>

<ol>
    <li>Нумерованный пункт</li>
</ol>

Что такое CSS?

CSS (Cascading Style Sheets) — каскадные таблицы стилей. Если HTML это скелет сайта, то CSS — это его одежда и макияж. С помощью CSS можно изменить цвет, шрифт, размер, расположение элементов и многое другое.

CSS состоит из селекторов и свойств:

селектор {
    свойство: значение;
    свойство: значение;
}

Пример:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

p {
    font-family: Arial, sans-serif;
    line-height: 1.6;
}

📌 Селектор — указывает, к какому тегу применить стиль (h1, p, .class, #id)

📌 Свойство — что именно изменить (color, font-size, margin...)

📌 Значение — значение свойства (blue, 16px, 10px...)

Основные свойства CSS

Вот самые популярные свойства, которые ты будешь использовать:

Текст и шрифт

/* Цвет и шрифт */
color: #ff0000;           /* Цвет текста */
font-size: 16px;          /* Размер шрифта */
font-family: Arial;      /* Шрифт */
font-weight: bold;       /* Жирный */
text-align: center;      /* Выравнивание */

/* Отступы между строками */
line-height: 1.5;

Фон и границы

/* Фон */
background: #f0f0f0;     /* Цвет фона */
background-image: url('bg.jpg'); /* Картинка */

/* Границы */
border: 1px solid black;
border-radius: 8px;       /* Скругление углов */

Отступы и размеры

/* Внешние отступы */
margin: 20px;             /* со всех сторон */
margin-top: 10px;         /* сверху */

/* Внутренние отступы */
padding: 15px;
padding-left: 10px;       /* слева */

/* Размеры */
width: 300px;            /* ширина */
height: 200px;           /* высота */

💡 margin — отступ снаружи элемента

💡 padding — отступ внутри элемента

Как подключить CSS к HTML?

Есть три способа добавить стили на страницу:

1. Встроенные стили (inline)

<p style="color: red; font-size: 18px;">Красный текст</p>

Не рекомендуется — трудно поддерживать.

2. Внутри тега style

<head>
    <style>
        p { color: blue; }
    </style>
</head>

3. Внешний файл (рекомендуется)

<head>
    <link rel="stylesheet" href="style.css">
</head>

✅ Внешний файл — самый удобный способ. Один файл style.css можно подключить ко всем страницам сайта.

Селекторы CSS

Селектор указывает, к каким элементам применить стили. Вот основные виды:

По тегу

p { color: black; }     /* все параграфы */
h1 { font-size: 32px; } /* все заголовки h1 */
div { margin: 10px; }   /* все div-ы */

По классу (class)

/* В HTML: <p class="intro">...</p> */
.intro { 
    font-size: 18px; 
    color: gray; 
}

По ID

/* В HTML: <div id="header">...</div> */
#header {
    background: #333;
    color: white;
}

Комбинирование

.menu-item { ... }        /* класс */
a.link { ... }            /* тег a с классом link */
#header .logo { ... }     /* элемент logo внутри header */

🔹 . — для классов

🔹 # — для ID

🔹 пробел — "внутри" (вложенность)

Блочная модель (Box Model)

Каждый элемент на странице — это прямоугольник. Он состоит из нескольких слоёв:

+---------------------------+
|         margin            |  ← внешний отступ
+---------------------------+
|         border            |  ← граница
+---------------------------+
|         padding           |  ← внутренний отступ
+---------------------------+
|                           |
|        content            |  ← содержимое (текст, картинка)
|                           |
+---------------------------+

Пример

.box {
    margin: 20px;      /* снаружи */
    border: 2px solid blue;
    padding: 15px;    /* внутри */
    width: 200px;     /* ширина контента */
    height: 100px;    /* высота контента */
}

box-sizing

/* По умолчанию width не включает padding и border */
* { box-sizing: border-box; }  /* исправляет это */

📦 Свойство box-sizing: border-box; очень полезно — включи его в начале CSS!

Flexbox

Flexbox — мощный инструмент для выравнивания элементов. Применяется к родительскому контейнеру:

Основные свойства

.container {
    display: flex;           /* включить flexbox */
    justify-content: center; /* по горизонтали */
    align-items: center;     /* по вертикали */
    gap: 10px;               /* отступ между элементами */
    flex-wrap: wrap;         /* перенос на новую строку */
}

justify-content (горизонталь)

flex-start   /* в начале */
flex-end     /* в конце */
center       /* по центру */
space-between /* равномерно (края прижаты) */
space-around  /* равномерно (края отступлены) */

align-items (вертикаль)

flex-start   /* в начале */
flex-end     /* в конце */
center       /* по центру */
stretch      /* растянуть (по умолчанию) */

Для дочерних элементов

.item {
    flex: 1;           /* занимает всё доступное место */
    flex-basis: 200px; /* базовая ширина */
    order: 1;          /* порядок (отрицательные — в начале) */
}

Позиционирование

Свойство position определяет, как элемент располагается на странице:

Основные значения

/* Статическое (по умолчанию) */
position: static;

/* Относительное — сдвиг относительно исходного места */
position: relative;
top: 10px; left: 20px;

/* Абсолютное — относительно ближайшего родителя */
position: absolute;
top: 0; right: 0;

/* Фиксированное — относительно окна браузера */
position: fixed;
top: 0; left: 0;

/* Липкое — прилипает при скролле */
position: sticky;
top: 10px;

Пример: всплывающее окно

.modal {
    position: fixed;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    background: white;
    padding: 20px;
    box-shadow: 0 0 10px rgba(0,0,0,0.5);
}

⚠️ absolute и fixed вырывают элемент из потока — другие элементы его не видят.

Адаптивный дизайн

Media queries позволяют менять стили в зависимости от размера экрана:

Базовая структура

/* Для всех экранов */
.container { width: 100%; }

/* Для экранов шире 768px */
@media (min-width: 768px) {
    .container { width: 750px; }
}

/* Для экранов шире 1024px */
@media (min-width: 1024px) {
    .container { width: 960px; }
}

Типичные брейкпоинты

/* Мобильные (портрет) */
@media (max-width: 480px) { ... }

/* Мобильные (альбом) / планшеты */
@media (max-width: 768px) { ... }

/* Ноутбуки */
@media (max-width: 1024px) { ... }

/* Большие мониторы */
@media (min-width: 1200px) { ... }

Пример: скрытие меню на мобильных

.menu { display: block; }

@media (max-width: 768px) {
    .menu { display: none; }
    .hamburger { display: block; }
}

📱 Всегда думай о мобильных пользователях — они часто составляют основной трафик!

CSS Grid

Grid — это двумерная сетка для сложных макетов. В отличие от Flexbox (строка ИЛИ колонка), Grid управляет строками И колонками одновременно:

Базовая структура

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* 3 колонки */
    grid-template-rows: auto;            /* автовысота */
    gap: 20px;                           /* отступ между ячейками */
}

Единицы измерения

grid-template-columns: 200px 200px 200px; /* фикс ширина */
grid-template-columns: 1fr 2fr 1fr;      /* пропорции (1:2:1) */
grid-template-columns: repeat(3, 1fr);    /* 3 колонки по 1fr */
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* адаптивно */

Растягивание ячеек

.item {
    grid-column: 1 / 3;  /* занимает колонки 1 и 2 */
    grid-row: 2 / 4;     /* занимает строки 2 и 3 */
}

Пример: карточки товаров

.products {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
    gap: 20px;
}

🔲 Grid идеален для карточек, галерей, макетов страниц. Flexbox — для компонентов (кнопки, меню, списки).

Анимации

CSS позволяет создавать плавные анимации без JavaScript:

transition — плавные переходы

.btn {
    background: blue;
    transition: background 0.3s, transform 0.2s;
}

.btn:hover {
    background: darkblue;
    transform: scale(1.05);
}

keyframe — сложные анимации

@keyframes bounce {
    0% { transform: translateY(0); }
    50% { transform: translateY(-20px); }
    100% { transform: translateY(0); }
}

.ball {
    animation: bounce 1s infinite;
}

Полезные свойства анимации

.element {
    animation-name: bounce;
    animation-duration: 1s;
    animation-timing-function: ease-in-out;
    animation-delay: 0.5s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

timing functions

ease        /* плавно (по умолчанию) */
linear      /* равномерно */
ease-in     /* ускорение в начале */
ease-out    /* замедление в конце */
ease-in-out /* ускорение + замедление */

🎬 Анимации делают интерфейс живым, но не переусердствуй — слишком много движения раздражает.

Формы и inputs

Формы — основа интерактивности. Без стилей выглядят уныло:

Базовая структура формы

Типы input

text, password, email, number, tel
date, time, color
checkbox, radio
file, hidden
range, search

Стилизация inputs

input, textarea, select {
    width: 100%;
    padding: 10px;
    border: 1px solid #ccc;
    border-radius: 5px;
    font-size: 16px;
}

input:focus {
    outline: none;
    border-color: blue;
    box-shadow: 0 0 5px rgba(0,0,255,0.3);
}

Чекбоксы и радио кнопки

/* Скрываем оригинал */
input[type="checkbox"] { display: none; }

/* Свой стиль */
input[type="checkbox"] + label::before {
    content: "☐";
    font-size: 20px;
}

input[type="checkbox"]:checked + label::before {
    content: "☑";
}

📝 Всегда используй label с for — это важно для доступности и удобства.

Псевдоклассы и псевдоэлементы

Псевдоклассы — особые состояния элемента. Псевдоэлементы — "виртуальные" части элемента:

Псевдоклассы (состояния)

/* При наведении */
:hover

/* При фокусе */
:focus

/* Активная ссылка */
:visited

/* Первый/последний ребёнок */
:first-child
:last-child
:nth-child(2)
:nth-child(odd)  /* нечётные */
:nth-child(even) /* чётные */

/* Первый/последний тип */
:first-of-type
:last-of-type

/* Пустой элемент */
:empty

/* Без класса */
:not(.class)

Псевдоэлементы

/* Перед элементом */
::before

/* После элемента */
::after

/* Первая буква */
::first-letter

/* Первая строка */
::first-line

/* Выделение текста */
::selection

Пример: иконка перед ссылкой

a[href^="http"]::before {
    content: "🔗 ";
}

a[href$=".pdf"]::before {
    content: "📄 ";
}

Пример: красивая кнопка

.btn::before {
    content: "";
    position: absolute;
    top: 0; left: 0;
    width: 100%; height: 100%;
    background: linear-gradient(45deg, transparent, rgba(255,255,255,0.3), transparent);
    transform: translateX(-100%);
    transition: transform 0.5s;
}

.btn:hover::before {
    transform: translateX(100%);
}

🎯 Псевдоклассы и псевдоэлементы — мощный инструмент для стилизации без изменения HTML.

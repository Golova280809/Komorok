Изучаем все основные теги и их применение

Что такое HTML?

HTML (HyperText Markup Language) — это язык разметки, на котором строятся все веб-страницы. Он определяет структуру содержимого: где заголовок, где параграф, где картинка или ссылка.

Теги и элементы

HTML использует теги — специальные команды в угловых скобках, которые создают элементы. Элемент — это тег плюс его содержимое.

<p>Это <em>параграф</em> текста.</p>
<h1>Заголовок страницы</h1>
  • Парные теги: Имеют открывающий (<p>) и закрывающий (</p>) тег. Содержимое находится между ними.
  • Одиночные теги (пустые теги): Не имеют закрывающего тега и не содержат содержимое. Например, <img> (изображение), <br> (перенос строки).

Атрибуты элементов

Атрибуты предоставляют дополнительную информацию об элементе и всегда находятся в открывающем теге. Они состоят из имени и значения (например, имя="значение").

<a href="https://example.com" target="_blank" title="Перейти на сайт">Пример ссылки</a>
  • href: указывает адрес ссылки.
  • target="_blank": открывает ссылку в новой вкладке.
  • title: всплывающая подсказка при наведении.

Блочные и строчные элементы

Это важнейшее различие в поведении элементов:

  • Блочные элементы: Занимают всю доступную ширину, каждый новый блочный элемент начинается с новой строки. Примеры: <div>, <p>, <h1>.
  • Строчные элементы: Занимают только ту ширину, которая необходима для их содержимого, и располагаются в одной строке с другими строчными элементами. Примеры: <span>, <a>, <strong>.

Глобальные атрибуты

Эти атрибуты могут быть использованы практически с любым HTML-элементом.

  • id="имя": Уникальный идентификатор элемента на странице.
  • class="имя-класса": Один или несколько классов, для стилизации или JS.
  • style="свойство: значение;": Встраиваемые CSS-стили (обычно не рекомендуется).
  • title="подсказка": Всплывающая подсказка при наведении мыши.
  • lang="ru": Язык содержимого элемента.
  • data-*="значение": Пользовательские атрибуты для хранения данных, доступных через JavaScript.
<div id="main-content" class="container dark-theme" lang="ru" data-user-id="123">
    <p title="Это основной текст">Привет, мир!</p>
</div>

Структура HTML-документа

Каждый HTML-документ имеет базовую структуру.

  • <!DOCTYPE html>: Объявление типа документа (HTML5). Всегда первая строка.
  • <html lang="ru">: Корневой элемент документа. Атрибут lang указывает язык.
  • <head>: Содержит метаданные документа (невидимые пользователю).
  • <body>: Содержит все видимое содержимое веб-страницы.
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <h1>Добро пожаловать!</h1>
</body>
</html>

Семантические теги для структуры страницы

HTML5 ввел множество тегов, которые помогают структурировать содержимое, объясняя его смысл браузерам и поисковикам (SEO).

  • <header>: Представляет вводную или навигационную часть секции или документа.
  • <nav>: Содержит навигационные ссылки.
  • <main>: Основное (уникальное) содержимое документа. На странице должен быть только один <main>.
  • <article>: Независимая, самодостаточная часть контента (например, статья в блоге, новостной пост).
  • <section>: Тематическая группировка содержимого.
  • <aside>: Содержимое, косвенно связанное с основным контентом (например, боковая панель, рекламный блок).
  • <footer>: Подвал секции или документа (информация об авторе, копирайт, ссылки).
  • <div>: Универсальный блочный контейнер без семантического значения (используется для группировки и стилизации).
  • <span>: Универсальный строчный контейнер без семантического значения (для стилизации части текста).
<body>
    <header><h1>Название Сайта</h1></header>
    <nav><a href="#">Меню</a></nav>
    <main>
        <section>
            <h2>Раздел</h2>
            <article><p>Статья...</p></article>
        </section>
        <aside><p>Сайдбар</p></aside>
    </main>
    <footer><p>© 2024</p></footer>
</body>

Теги для работы с текстом

Эти теги используются для структурирования и форматирования текстового контента.

Заголовки и параграфы

  • <h1> - <h6>: Заголовки различных уровней (от самого важного до наименее важного).
  • <p>: Параграф (абзац) текста.
  • <br>: Перенос строки (разрыв).
  • <hr>: Горизонтальная линия (разделитель тем).
<h1>Самый большой заголовок</h1>
<h3>Подзаголовок</h3>
<p>Это первый параграф.<br>Текст после переноса.</p>
<hr>
<p>Это второй параграф.</p>

Базовое форматирование текста

  • <b>: Жирный текст (без дополнительного акцента, просто выделение).
  • <i>: Курсивный текст (без дополнительного акцента, для альтернативного голоса/настроения).
  • <strong>: Жирный текст с сильным акцентом (важный).
  • <em>: Курсивный текст с акцентом (выделение).
  • <mark>: Выделенный текст (как маркером).
  • <small>: Мелкий текст (для примечаний, копирайтов).
  • <del>: Зачеркнутый текст (удаленный).
  • <ins>: Подчеркнутый текст (вставленный).
  • <sub>: Подстрочный индекс (H<sub>2</sub>O).
  • <sup>: Надстрочный индекс (x<sup>2</sup>).
<p><b>Жирный</b> и <i>курсивный</i> текст.</p>
<p>Это <strong>важно</strong>, а это <em>акцент</em>.</p>
<p>Я <mark>выделен</mark>. <small>Мелкий шрифт.</small></p>
<p><del>Старая цена</del> <ins>Новая цена</ins></p>
<p>Вода H<sub>2</sub>O, формула E=mc<sup>2</sup>.</p>

Семантические теги для текста

  • <blockquote>: Для длинных цитат, обычно с отступом.
  • <q>: Для коротких цитат (браузеры обычно добавляют кавычки).
  • <abbr>: Аббревиатура или акроним. Атрибут title показывает полное значение.
  • <cite>: Название произведения (книга, фильм, песня).
  • <code>: Фрагмент компьютерного кода.
  • <pre>: Предварительно форматированный текст (сохраняет пробелы и переносы).
  • <address>: Контактная информация для автора/владельца документа.
  • <time>: Дата или время. Может иметь атрибут datetime.
<blockquote>
    <p>"Многозначность — признак плохого дизайна."</p>
    <cite>Дональд Норман</cite>
</blockquote>
<p>Как сказал поэт, <q>я вас любил</q>.</p>
<p>Изучаю <abbr title="HyperText Markup Language">HTML</abbr>.</p>
<p>Команда для вывода: <code>console.log("Hello");</code></p>
<pre>
Это
    предварительно
        отформатированный
            текст.
</pre>
<address>
    Автор: Иван Иванов<br>
    Email: <a href="mailto:ivan@example.com">ivan@example.com</a>
</address>
<p>Опубликовано <time datetime="2024-04-07">7 апреля 2024</time>.</p>

Списки

HTML предлагает несколько типов списков для организации информации.

  • <ul>: Неупорядоченный (маркированный) список.
  • <ol>: Упорядоченный (нумерованный) список.
  • <li>: Элемент списка (используется внутри <ul> или <ol>).
  • <dl>: Список определений.
  • <dt>: Термин для определения.
  • <dd>: Описание термина.
<ul>
    <li>Пункт 1</li>
    <li>Пункт 2</li>
</ul>

<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
</ol>

<dl>
    <dt>HTML</dt>
    <dd>Язык разметки гипертекста.</dd>
    <dt>CSS</dt>
    <dd>Каскадные таблицы стилей.</dd>
</dl>

Ссылки и Изображения

Основа навигации и визуального контента в вебе.

  • <a>: Ссылка. Атрибут href обязателен. target="_blank" откроет в новой вкладке.
  • <img>: Изображение. Атрибуты src (путь к файлу) и alt (альтернативный текст для SEO и доступности) обязательны.
  • <figure>: Самодостаточный контент (изображение, диаграмма, код).
  • <figcaption>: Подпись для <figure>.
<a href="https://ya.ru">Перейти на Яндекс</a>

<img src="path/to/image.jpg" alt="Описание изображения" width="300" height="200">

<figure>
    <img src="path/to/another-image.png" alt="Еще одно изображение">
    <figcaption>Это подпись к изображению.</figcaption>
</figure>

Таблицы

Используются для отображения табличных данных, а не для создания макета страницы.

  • <table>: Контейнер для всей таблицы.
  • <caption>: Заголовок таблицы.
  • <thead>: Секция заголовков таблицы.
  • <tbody>: Секция основного содержимого таблицы.
  • <tfoot>: Секция футера таблицы.
  • <tr>: Строка таблицы.
  • <th>: Заголовок ячейки таблицы (обычно жирный и по центру).
  • <td>: Обычная ячейка таблицы.
  • Атрибуты colspan и rowspan: Объединяют ячейки по горизонтали и вертикали.
<table>
    <caption>Список товаров</caption>
    <thead>
        <tr>
            <th>Название</th>
            <th>Цена</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Монитор</td>
            <td>15000 руб.</td>
        </tr>
        <tr>
            <td>Клавиатура</td>
            <td>3000 руб.</td>
        </tr>
    </tbody>
</table>

Формы

Используются для сбора данных от пользователя.

  • <form>: Контейнер для всех элементов формы. Атрибуты action (куда отправить данные) и method (GET/POST).
  • <input>: Основной элемент ввода. Атрибут type определяет вид (text, password, email, number, checkbox, radio, submit, file и др.).
  • <label>: Метка для элемента формы (улучшает доступность). Атрибут for связывает с id элемента.
  • <textarea>: Многострочное текстовое поле.
  • <select>: Выпадающий список.
  • <option>: Элемент внутри <select>.
  • <button>: Кнопка. Атрибут type (submit, reset, button).
  • <fieldset>: Группирует связанные элементы формы.
  • <legend>: Заголовок для <fieldset>.
<form action="/submit-form" method="post">
    <fieldset>
        <legend>Личные данные</legend>
        <label for="name">Имя:</label>
        <input type="text" id="name" name="user_name" required><br><br>

        <label for="email">Email:</label>
        <input type="email" id="email" name="user_email"><br><br>

        <label for="message">Сообщение:</label><br>
        <textarea id="message" name="user_message" rows="5"></textarea><br><br>

        <label for="country">Страна:</label>
        <select id="country" name="user_country">
            <option value="ru">Россия</option>
            <option value="us">США</option>
        </select><br><br>

        <button type="submit">Отправить</button>
    </fieldset>
</form>

Медиа-элементы

Для встраивания аудио и видео контента.

  • <audio>: Встраивает аудио. Атрибуты: src, controls (показывает элементы управления), autoplay, loop, muted.
  • <video>: Встраивает видео. Атрибуты: src, controls, autoplay, loop, muted, poster (изображение-заглушка), width, height.
  • <source>: Позволяет указать несколько медиа-ресурсов для <audio> или <video>.\
  • <track>: Субтитры, описания, метаданные для <video>.
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Ваш браузер не поддерживает аудио.
</audio>

<video width="320" height="240" controls poster="poster.jpg">
    <source src="video.mp4" type="video/mp4">
    Ваш браузер не поддерживает видео.
</video>

Интерактивные элементы

Теги для создания интерактивных элементов на странице без JavaScript (в простых случаях).

  • <details>: Создает виджет раскрытия, в котором пользователь может открыть или закрыть дополнительную информацию.
  • <summary>: Предоставляет видимый заголовок для <details>.
  • <dialog>: Создает диалоговое окно или всплывающее окно (требует JS для открытия/закрытия в большинстве случаев).
<details>
    <summary>Нажмите, чтобы узнать больше</summary>
    <p>Это скрытый контент, который раскрывается по клику.</p>
</details>

<!-- Для работы <dialog> обычно нужен JavaScript -->
<dialog id="myDialog">
    <p>Это диалоговое окно.</p>
    <button onclick="document.getElementById('myDialog').close()">Закрыть</button>
</dialog>
<button onclick="document.getElementById('myDialog').showModal()">Открыть диалог</button>

Прочие и менее используемые теги

  • <iframe>: Встраивает другой HTML-документ (или страницу) в текущий.
  • <canvas>: Используется для рисования графики с помощью JavaScript.
  • <svg>: Встраивает масштабируемую векторную графику (XML-разметка).
  • <template>: Содержит контент, который не отображается при загрузке страницы, но может быть использован JavaScript.
  • <noscript>: Контент, который отображается, если JavaScript отключен или не поддерживается.
<iframe src="another-page.html" width="600" height="400"></iframe>

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000;"></canvas>

<template id="myTemplate">
    <p>Это шаблонный контент.</p>
</template>

<noscript>
    <p>Пожалуйста, включите JavaScript.</p>
</noscript>

Устаревшие теги (Не рекомендуется к использованию)

Эти теги были удалены из стандарта HTML5 или их функции лучше выполнять с помощью CSS.

  • <font>: Для изменения шрифта, цвета, размера (используйте CSS: font-family, color, font-size).
  • <center>: Для центрирования содержимого (используйте CSS: text-align: center; или margin: 0 auto;).
  • <big> / <small> (многократное): Для увеличения/уменьшения текста (используйте CSS: font-size). Тег <small> остался для семантического значения "мелкий текст" (например, копирайт).
  • <strike>: Зачеркнутый текст (используйте <del> или CSS text-decoration: line-through;).
  • <frame>, <frameset>, <noframes>: Для создания фреймов (используйте <iframe> или CSS-макеты).

Всегда стремитесь использовать современные семантические теги и стилизовать элементы с помощью CSS!
